About

Printify started as my final project submission for Harvard's CS50x course and is still under further development.

For my project, I wanted to challenge myself to build a web app that calls on an API. Around the same time, I came across the practice of generative art/ creative coding and was inspired to create art from data obtained from Spotify's Web API.

These two things culminated in the development of Printify, a web app that generates art prints from Spotify's Audio API. I had a lot of fun making this app, and I hope you have fun using it too!
